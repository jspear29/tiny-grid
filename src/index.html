configuration:

$col-slug: 'col' !default;
defines the prefix for columns. this site uses meow. eg. meow-xs-12
$row-slug: 'row' !default;
defines class for row
$container-width: 100 !default;
default container width
custom media queries! $media-sizes is map of size:px values to define what kind of columns get generated and what max/min widths get applied. use sass map functions to add, merge, or totally override them!
